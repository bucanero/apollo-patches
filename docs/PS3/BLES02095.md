### 5. Update CRC32:0 for SYS-DATA (Required)

Target File: `SYS-DATA`

```
set pointer:eof+1
set range:0x000004,pointer
set [hash]:CRC32:0
set [hash]:xor:FFFFFFFF
write at 0x000000:[hash]
```
