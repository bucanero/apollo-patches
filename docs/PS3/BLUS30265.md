# BLUS30265.savepatch

### 1. Update CRC32 for SF4-SAVE.DAT (Required)

Target File: `SF4-SAVE.DAT`

```
set pointer:eof+1
set range:0x000004,pointer
set [hash]:CRC32
write at 0x000000:[hash]
```
