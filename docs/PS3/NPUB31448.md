### 2. Update Settings for SAVE.DAT

Target File: `SAVE.DAT`

```
search "Settings"
set [begin]:pointer+0x20
set [size]:read([begin],4)
set [end_of_range]:[begin]+[size]-1
set [begin]:[begin]+4
set range:[begin],[end_of_range]
set [hash]:CRC32
set pointer:[end_of_range]+1
write next (0):[hash]
```
