### 2. Update CRC32 for LOTR.DAT (Required)

Target File: `LOTR.DAT`

```

set range:0x000000,pointer
set [hash]:CRC32
set [hash]:xor:FFFFFFFF
set pointer:eof-3
write next (0):[hash]
```
