#  Street Fighter X Tekken 

`BLES01252.savepatch`

### 1. Unlock All Battle Titles

Target File: `SAVEDATA`

```
80010004 55544600
4A000015 FFFFFFFF
40130004 00000000
```

### 2. Unlock All EX Battle Titles

Target File: `SAVEDATA`

```
80010004 55544600
4A00006D FFFFFFFF
40030004 00000000
```

### 3. Update crc32 (required)

Target File: `SAVEDATA`

```
set range:0x4,0x11E676
set [crc]:crc-32
write at 0x0:[crc]
```
