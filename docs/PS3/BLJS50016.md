#  Sacred 2: Fallen Angel [PlayStation 3 the Best] 

`BLJS50016.savepatch`

### 1. Decompress Save (Required)

Target File: `SAVE.DAT`

```
Decompress SAVE.DAT
```

### 2. 32767 Skill Points(Apply on xxx.neo file)

Target File: `~extracted\*.neo`

```
search 0xFFFFAA07
write next 8:0x7FFF
```

Target File: `~extracted\*.neo`

```

```

```

```

