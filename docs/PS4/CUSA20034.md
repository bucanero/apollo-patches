# The Legend of Heroes: Trails of Cold Steel IV

`CUSA20034.savepatch`

### 1. Unpack Save data (Required)

Target File: `user.dat`

```
import loh_trails
unpack = loh_trails.loh_trails_decompress(savedata)
savedata = unpack
```

### 2. Fix custom CRC32 (Required)

Target File: `user.dat`

```
import loh_trails
import ubinascii
trails_crc = loh_trails.loh_trails_crc32(savedata, 0x10)
print("New CRC:", ubinascii.hexlify(trails_crc))
savedata[12:16] = trails_crc
```

### 3. Compress Save data (Required)

Target File: `user.dat`

```
import loh_trails
compressed = loh_trails.loh_trails_compress(savedata)
savedata = compressed
```
