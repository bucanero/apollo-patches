### 1. Decrypt Save data (Required)

Target File: `data*.bin`

```

endian_swap(4)
decrypt blowfish_cbc("K<>$cl%isqA|~nV4W5~3z_Q)j]5DHdB9sb{cI9Hn&Gqc-zO8O6zf", 0x0000000000000000)
endian_swap(4)
set range:0x0010,0x001F
endian_swap(4)
encrypt blowfish_cbc("K<>$cl%isqA|~nV4W5~3z_Q)j]5DHdB9sb{cI9Hn&Gqc-zO8O6zf", 0x0000000000000000)
decrypt blowfish_ecb("K<>$cl%isqA|~nV4W5~3z_Q)j]5DHdB9sb{cI9Hn&Gqc-zO8O6zf")
endian_swap(4)
```

Target File: `data*.bin`

```

set [hash]:endian_swap
```
